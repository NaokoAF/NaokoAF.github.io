# what is it
neon gyro is a mod for [Neon White](https://store.steampowered.com/app/1533420/Neon_White/), released shortly after
my previous mod [Hasty Controls](/projects/hasty-controls), and also adds gyro aiming in a very similar way.

![mod showcase](../assets/neon-gyro-showcase.webp)

# how it came to be
while talking to ivan from [FlickStick Videos](https://www.youtube.com/@FlickStickVids), he casually made a request for
a mod like this, and i whipped something up in a few hours by borrowing a lot of the code from hasty controls.

there's really not a lot to say about this one. the core of the code is a single hook into the game's camera, which
adds gyro and flick stick on top of the existing controls. thanks to my previous work on
[GyroHelpers](https://github.com/NaokoAF/GyroHelpers), it was as easy as slotting it in and writing some glue.
